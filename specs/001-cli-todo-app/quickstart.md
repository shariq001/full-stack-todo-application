# Quickstart Guide: CLI Todo App

## Prerequisites
- Python 3.13+ installed
- UV package manager installed

## Setup
1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: `uv sync` (or `pip install -r requirements.txt`)

## Running the Application
1. Execute: `python src/main.py`
2. The application will start in interactive mode
3. Follow the on-screen menu options to manage your tasks

## Available Commands
- `1` or `add`: Add a new task
- `2` or `view`: View all tasks
- `3` or `update`: Update an existing task
- `4` or `delete`: Delete a task
- `5` or `complete`: Mark a task as complete
- `6` or `incomplete`: Mark a task as incomplete
- `q` or `quit`: Exit the application

## Example Usage
1. Run the application: `python src/main.py`
2. Select option `1` to add a task
3. Enter the task title and description when prompted
4. Use option `2` to view your tasks
5. Continue managing your tasks as needed
6. Type `q` to quit the application

## Troubleshooting
- If you encounter any errors, ensure Python 3.13+ is installed
- Verify all required dependencies are installed
- Check that you're running the application from the project root directory
